# JPirates
Licensed players are good, but what about pirates? Add the pirates you approve of by nickname.
## Requirements:
- https://github.com/mitriyf/FastLogin/releases/download/1.0/FastLoginBukkit.jar | Download the plugin so that JPirates can work.
## Optional requirements:
- AuthMe Reloaded (5.X)
- xAuth
- CrazyLogin
- LoginSecurity
- AdvancedLogin (Paid)
- AlixSystem
- UltraAuth
- BungeeAuth (BungeeCord/Waterfall plugin)
- and others from: https://www.spigotmc.org/resources/fastlogin.14153/
